## Theme

Efferd UI uses CSS variables for theming. You can customize the colors by modifying the variables in your `globals.css` file.

### Color Variables

The theme is defined using OKLCh color values for better perceptual uniformity:

```css
:root {
  --background: oklch(1 0 0);
  --foreground: oklch(0.145 0 0);
  --primary: oklch(0.205 0 0);
  --primary-foreground: oklch(0.985 0 0);
  --secondary: oklch(0.97 0 0);
  --muted: oklch(0.97 0 0);
  --accent: oklch(0.97 0 0);
}
```

### Dark Mode

Dark mode is supported through the `.dark` class. The theme automatically switches based on system preferences or user selection:

```css
.dark {
  --background: oklch(0.145 0 0);
  --foreground: oklch(0.985 0 0);
  --primary: oklch(0.922 0 0);
}
```

## Typography

Efferd UI uses the following fonts:

| Font   | Usage    | Variable            |
| ------ | -------- | ------------------- |
| Geist  | Body     | `--font-geist-sans` |
| Outfit | Headings | `--font-heading`    |

## Border Radius

The global border radius is controlled by the `--radius` variable:

```css
:root {
  --radius: 0.65rem;
}
```

All components derive their border radius from this value using `calc()`:

- `--radius-sm`: `calc(var(--radius) - 4px)`
- `--radius-md`: `calc(var(--radius) - 2px)`
- `--radius-lg`: `var(--radius)`
- `--radius-xl`: `calc(var(--radius) + 4px)`
